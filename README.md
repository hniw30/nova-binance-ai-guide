## What is NOVA?
NOVA is a floating AI widget that lives in the corner of Binance — multilingual, 
24/7, powered by Claude Sonnet + 12 Binance AI Agent Skills.

**8 Features:** AI Chat · Voice Call · Coin Intelligence (12 AI Skills) · 
Web3 Guide · Live Dubbing · Fake News Detector · KOL Feed · Smart Reminders

**4 Languages:** English · Tiếng Việt · 中文 · Türkçe


## How to Run Locally
1. Download `NOVA_ultimate.html`
2. Open in **Google Chrome** (required for Voice feature)
3. Internet connection needed (Claude API calls)
4. No installation required
